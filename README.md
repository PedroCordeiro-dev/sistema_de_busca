# 🍎 Sistema De Busca De Frutas

> **Meu primeiro CRUD em Python:** Um sistema prático para busca e gestão de inventário de frutas.

Este projeto marca meu início nos conceitos de **CRUD** (Create, Read, Update, Delete). Desenvolvi uma interface de linha de comando focada em usabilidade simples e manipulação de dados em listas.

---

### 🎯 O que o sistema faz?

*   🔍 **Buscar:** Localiza frutas específicas no inventário.
*   📋 **Listar:** Exibe todos os itens cadastrados de forma organizada.
*   ➕ **Adicionar:** Adiciona novas frutas ao sistema.
*   ✏️ **Alterar:** Permite alterar nomes de frutas já existentes.
*   🗑️ **Remover:** Exclui itens do inventário.
*   🚪 **Sair:** Finaliza a execução do programa com segurança.

---

### 🛠️ Tecnologias & Conceitos

*   **Linguagem:** Python 3.14
*   **Lógica Aplicada:** Estruturas de repetição (`while`), condicionais (`if/else`) e manipulação de listas.

---

### 🚀 Como testar

1. Certifique-se de ter o **Python** instalado.
2. Baixe o arquivo `main.py`.
3. No terminal, execute:

```bash
python main.py
```

---

### 🧠 O que aprendi neste projeto:

Desenvolver este sistema me ajudou a entender como estruturar um menu interativo, tratar de erros que podem acontecer e a lidar com estruturas de repetição.

Além disso, implementei o algoritmo de **Pesquisa Binária** (aprendido no livro *'Entendendo Algoritmos'*) na função de busca. Isso permitiu que o sistema localize frutas de forma muito mais eficiente ($O(\log n)$), em vez de percorrer a lista item por item.
